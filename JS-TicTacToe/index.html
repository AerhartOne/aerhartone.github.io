Tic-Tac-Toe

It's the classic game everyone knows, and eventually figures out they can't win.

In the interests of saving time (and because coding game AIs can be a pain lol), your opponent for this game plays defensively. So you shouldn't get stuck in an infinite string of drawn games.

The game is configurable by changing Game Settings variables in the tictactoe.js file. Settings include the following:

  • Grid size
  • Number of symbols in a row to win
  • Number of players
  • Which players will be AI-controlled
  • Player symbols
